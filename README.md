# Youth in AI Workshop: Speech Recognition Presentation

An open-source, hands-on HTML presentation for teaching speech recognition to non-technical audiences.

The deck explains:

- what speech recognition is
- how audio becomes text
- spectrograms and pattern matching
- modern AI speech models
- real-life applications
- platform demos with tools like ElevenLabs, OpenAI Audio, Deepgram, AssemblyAI, Google Speech-to-Text, and Azure Speech
- practical activities for a 40-minute workshop

## Open Locally

Open `index.html` in a browser.

## Deploy

This is a static site and can be deployed on Vercel, GitHub Pages, Netlify, or any static hosting provider.
